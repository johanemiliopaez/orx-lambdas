# Lambda Environment Variables

Configura la función Lambda con las siguientes variables de entorno para conectarse a la base de datos PostgreSQL:

- `PGHOST`: host o endpoint del clúster (por ejemplo, `your-db-host.amazonaws.com`).
- `PGPORT`: puerto del servicio (por defecto `5432`).
- `PGDATABASE`: nombre de la base de datos.
- `PGUSER`: usuario con permisos sobre la tabla `app_user`.
- `PGPASSWORD`: contraseña del usuario anterior.
